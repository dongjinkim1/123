# MBTS 학문 5원칙

### 원칙 1: 경향론 (Non-Determinism)
> 사주는 경향과 가능성을 보여주며, 결정하지 않는다.

### 원칙 2: 가설 정직성 (Hypothesis Honesty)
> 모든 매핑은 근거 강도(확립/유력/탐색)를 명시한다.

### 원칙 3: 이중 독립성 (Dual Independence)
> 사주학과 심리학은 각자의 체계를 유지한다. MBTS는 둘의 "대화"이지 "합병"이 아니다.

### 원칙 4: 성장 지향 (Growth Orientation)
> 모든 분석은 진단(50%)과 성장 처방(50%)으로 구성된다. 처방은 구체적 행동을 포함.

### 원칙 5: 자기 검증 (Self-Verification)
> 모든 풀이는 "맞지 않는 부분"을 찾도록 유도한다. 불일치는 더 깊은 통찰의 시작점.

# PART I. 이론적 기반

## 1. MBTS의 핵심 전제 🟢확립

사주와 MBTI는 같은 현실을 다른 방향에서 본다.
- **사주**: 생년월일시(객관적 데이터) → 성격 패턴
- **MBTI**: 자기보고 설문(주관적 데이터) → 성격 패턴

| 체계 | 한계 | MBTS의 해결 |
|------|------|-------------|
| MBTI | 변화를 설명 못 함 | 대운/세운으로 시간에 따른 변화 설명 |
| 사주 | 전문용어 장벽 | MBTI 인지기능이라는 보편 언어로 번역 |
| 둘 다 | 서로 연결 안 됨 | MBTS가 통합 프레임워크 제공 |

**MBTS = 시간축이 있는 동적 성격 분석 체계**

## 2. MBTS 10기능 체계 — 천간 ↔ 인지기능 매핑 🟡유력

> **[가설 명시]** 아래 매핑은 구조적 대응 + 경험적 관찰에 기반한 **검증 대기 가설**이다.
> 통계적 검증은 Phase 2에서 수행 예정. 검증 결과에 따라 수정되거나 폐기될 수 있다.

**오행은 에너지의 "종류"를, 음양은 "방향"을 결정한다.**

| 오행 | 에너지 | 양 (외향) | 음 (내향) |
|------|--------|-----------|-----------|
| 木 | 직관 | **甲 = Ne** (가능성 탐색) | **乙 = Ni** (깊은 통찰) |
| 火 | 감정 | **丙 = Fe** (사회적 조화) | **丁 = Fi** (개인 가치관) |
| **土** | **통합** | **戊 = Ce** (외향 통합 — 구조화/안정화) *[MBTS 고유]* | **己 = Ci** (내향 통합 — 수용/포용) *[MBTS 고유]* |
| 金 | 사고 | **庚 = Te** (효율/시스템) | **辛 = Ti** (논리적 프레임) |
| 水 | 감각 | **壬 = Se** (현재 경험) | **癸 = Si** (기억/디테일) |

> **Ce/Ci 정의:** 土는 MBTI의 J/P축이 아니다(J/P는 기능이 아닌 태도 — 범주 오류).
> 대신 MBTS 고유 기능으로 정의: Ce = 외부 세계를 구조화하는 중심 잡기, Ci = 내부 세계를 수용하는 유연한 포용.

**구조적 근거 (은유가 아닌 구조적 대응):**
1. 방향 일치: 오행의 양=외향, 음=내향 / 인지기능의 e=외향, i=내향
2. 상생상극: 木生火 = 직관이 감정을 촉진 → 심리학에서도 직관 발달이 감정 통찰을 깊게 함
3. 독립성: 오행 5가지와 인지기능 축이 모두 서로 환원 불가능한 독립 범주

### 2-1. 천간충 = 반대 기능의 대결 🟡유력

| 충 | 기능 충돌 | 심리 현상 |
|----|----------|-----------|
| 甲庚 | Ne vs Te | 가능성 vs 효율 |
| 乙辛 | Ni vs Ti | 직관 vs 분석 |
| 丙壬 | Fe vs Se | 조화 vs 경험 |
| 丁癸 | Fi vs Si | 가치관 vs 기억 |

### 2-2. 천간합 = 기능 융합 🟡유력

| 합 | 합화 결과 | MBTS 의미 |
|----|----------|-----------|
| 甲己 | → 化土 (Ce/Ci) | 탐색+수용 → 통합 에너지 탄생 |
| 乙庚 | → 化金 (Te/Ti) | 통찰+효율 → 사고 에너지 탄생 |
| 丙辛 | → 化水 (Se/Si) | 조화+분석 → 감각 에너지 탄생 |
| 丁壬 | → 化木 (Ne/Ni) | 가치관+경험 → 직관 에너지 탄생 |
| 戊癸 | → 化火 (Fe/Fi) | 구조+기억 → 감정 에너지 탄생 |

### 2-3. 대운 간합에 의한 기능 변환 (Phase Transition) 🟡유력

대운 천간이 원국 천간과 합하면, 기존 기능이 **변환**될 수 있다.

| 합화 성공 | 합화 실패 (합거) |
|-----------|----------------|
| 기존 기능이 새 기능으로 건강하게 변환. 성장 | 기존 기능이 묶여서 약화. 억압 |
| 조건: 월지가 합화 오행을 지지 | 조건: 합화의 조건 불충족 |

### 2-4. 투합/쟁합 = 내면의 삼각관계 🔵탐색

| 구조 | MBTS 의미 |
|------|-----------|
| 투합 (甲甲己) | 하나의 자원을 두 기능이 경쟁 |
| 쟁합 (甲己己) | 하나의 기능에 과도한 에너지가 달라붙어 압도 |

# PART III. 에너지 역학 시스템

## 6. 십성 = 인지기능 간 역학관계 🟢확립

### 십성 비중 정밀 측정 (코드에 구현됨)

5그룹 비중이 **소수점 첫째자리**까지 산출됨:
```
십성비중: 비겁=2.7 식상=1.0 재성=0.7 관성=1.4 인성=2.2
```
- 천간 십성: 1.0점
- 지지 정기 십성: 0.7점 (가중치)
- 이 비중이 "어떤 에너지가 얼마나 강한가"를 수치로 보여줌

### 개별 십성 카운트 (10개 각각)
비견/겁재/식신/상관/편재/정재/편관/정관/편인/정인 각각의 개수를 별도 집계.
같은 "식상 2.0"이라도 "식신 2 + 상관 0"과 "식신 0 + 상관 2"는 완전히 다른 풀이.

| 십성 | MBTS 대응 |
|------|-----------|
| **비겁** | 주기능 강화. 독립적이지만 고집스러울 경향 |
| **식상** | 창의적 출구. 표현/아이디어 에너지 |
| **재성** | 현실 적응. 돈/기회를 잡는 실행력 경향 |
| **관성** | 도전/스트레스. 외부 압박 경향 |
| **인성** | 회복/학습. 주기능을 키워주는 에너지 경향 |

## 7. 5신 체계 = 심리적 에너지 처방전 🟢확립

| 5신 | MBTS 대응 |
|-----|-----------|
| **용신** | 가장 필요한 에너지. "이걸 개발하면 인생이 바뀔 가능성" |
| **희신** | 용신을 활성화시키는 보조 |
| **기신** | 과잉일 때 독이 되는 기능. "자제하면 좋을 영역" |
| **구신** | 방해 경향을 강화. 경계 대상 |
| **한신** | 상황에 따라 도움/방해 |

> **용신 = 매슬로 욕구단계에서 가장 결핍한 단계를 채우는 열쇠**와 유사 구조 🔵탐색

## 8. 오행 밸런스 = 인지기능 밸런스 🟢확립

### 이중 카운팅 시스템 (코드에 구현됨)

오행 개수를 **두 가지 레이어**로 측정하여 AI에 전달:

| 레이어 | 산출 방식 | 의미 |
|--------|----------|------|
| **표면 8자** (`saju.el`) | 천간4 + 지지4의 오행만 카운트 | 겉으로 드러나는 에너지 분포 |
| **지장간 포함** (`saju.elFull`) | 표면 + 지장간 여기/중기/정기 가중합산 | 숨겨진 잠재력을 포함한 실제 에너지 분포 |

> 풀이 핵심: "표면 火=0이지만 지장간 포함 火=1.4 → 겉으로 감정 표현이 안 보이지만, 속에 감정 에너지가 잠재되어 있는 구조. 대운에서 火가 오면 폭발적으로 터질 수 있음"

## 10. 신강/신약 = 자아 강도 (5단계 정밀 체계) 🟢확립

자아 에너지(비겁+인성) vs 환경 에너지(식상+재성+관성)의 **비율**로 판별.
코드: `gg.selfStr / (gg.selfStr + gg.otherStr)` → 퍼센트 산출

## 10-1. MBTI 강도 = 축별 3단계 정밀 프로필 🟢확립

MBTI 4글자(예: INFP)만으로는 부족. 같은 INFP라도 **"I가 51%"와 "I가 95%"는 완전히 다른 사람**.

## 36. 진태양시 보정 시스템 🟢확립

사주 계산의 정확도를 결정하는 핵심. 출생 시간을 **진태양시(True Solar Time)**로 보정.

- `equationOfTime()` — 균시차 계산 (지구 공전 타원/자전축 기울기 보정)
- `getTrueSolarCorrection()` — 경도 기준 보정 (서울=127° 기준, 사용자 출생지 경도 반영)
- 최대 ±16분 보정 → 시주(時柱)가 바뀔 수 있음 → 풀이가 완전히 달라짐

### 36-1. 시간 정밀도 원리 🟢확립

> MBTS는 **분(分) 단위 + 출생지 경도 보정** 분석을 수행한다.
> 이는 기존 유형론 대비 극도로 높은 해상도를 제공:
>
> - MBTI: 16유형
> - MBTS(사주4주+MBTI+강도): 이론적으로 **수억 개**의 고유 프로필
>
> **같은 날 태어난 쌍둥이도 다른 이유:**
> 출생 시간 차이(보통 5~30분) → 진태양시 보정 시 시주가 바뀔 수 있음
> 시주가 같아도 MBTI 강도(55/68/88)가 다를 수 있음

### 39-1. 물상의 학문적 지위 — 체현적 인지 이론 🟡유력

> **MBTS 물상론:** 물상(物象)은 단순 비유(metaphor)가 아닌, 오행 에너지의 **체현적 인지 형태(Embodied Cognitive Form)**이다.
> 인간의 추상적 사고가 신체적·물리적 경험에 기반한다는 체현적 인지 이론(Lakoff & Johnson, 1980)에 의해 학문적으로 방어됨.
> 甲木이 "큰 나무"인 것은 문화적 약속이 아니라, 甲의 에너지 특성(상승·확장·유연한 강함)을 인간이 보편적으로 체현 인지하는 형태.

### 39-2. 원국 풍경화 이론 (Natal Landscape Theory) 🟡유력

사주 원국은 **3층 물상의 조합으로 하나의 풍경화(Landscape)**를 형성한다:

| 층위 | 물상 소스 | 의미 |
|------|----------|------|
| **주인공** | 일간 천간 물상 | 에너지의 본질적 형태 |
| **땅** | 일지 지지 물상 | 무의식적 기반 |
| **계절** | 월지 계절 물상 | 시간적 맥락 |

> "겨울 바다 위의 촛불" = 문학적 장식이 아니라 에너지 구조의 체현적 서술.
> 풍경 내의 **조화/부조화**가 심리적 안정/갈등과 대응함. (심상 작업 Imagery Work과 유사)

### 39-3. 물상 조화도 (Landscape Harmony Index) 🔵탐색

일간 물상과 월지 계절의 **에너지 적합성**:

| 조합 | 조화도 | 심리적 의미 |
|------|--------|------------|
| 火일간 + 여름 | 과잉 조화 | 에너지 폭발. 강렬하지만 소진 빠름 |
| 火일간 + 겨울 | 부조화 | 내적 에너지와 환경의 괴리. 복잡한 내면 |
| 水일간 + 겨울 | 자연 조화 | 환경이 에너지를 지지. 안정적 |
| 金일간 + 봄 | 약한 부조화 | 에너지가 환경에서 약화. 보충 필요 |

> 부조화 자체가 나쁜 것이 아님 — 오히려 창의성과 깊이가 나올 수 있음.

## 41. 암합(暗合) 시스템 🟡유력

지장간끼리의 숨겨진 합. 겉으로는 안 보이지만 속에서 작동하는 인연.

- `saju.amhap` 배열에 저장: `{from, to, hapOh, gungwi}`
- AI 프롬프트에 전달: "★암합(숨겨진 합): 월지↔일지=합화목 [직업↔배우자 궁 숨겨진 인연]"
- 풀이: "겉으로는 직업과 연애가 관련 없어 보이지만, 속에서는 연결되어 있는 구조"

### 41-1. 암합의 3층위 이론 🔵탐색

> 암합(暗合)은 지장간의 **어떤 층위**에서 발생하느냐에 따라 의식화 정도가 다르다.
>
> | 암합 층위 | 의식화 정도 | 심리적 의미 |
> |-----------|-----------|------------|
> | **정기×정기** | 반의식 | "왜인지 모르게 끌리는" 수준. 인식하면 빠르게 의식화 가능 |
> | **정기×중기** | 약한 무의식 | "나도 모르게 반복되는 패턴". 타인이 지적하면 인식 가능 |
> | **여기×여기** | 깊은 무의식 | "존재 자체를 모르는 연결". 장기간 관찰로만 발견 |

### 41-2. 암합 의식화 경로 🔵탐색

> 암합 → 투출 → 밝합 = **"무의식적 연결이 의식화되어 선택 가능해지는 과정"**
> - 의식화 시점: 대운/세운에서 해당 천간이 등장할 때
> - 의식화되면: 그 연결을 강화할지 끊을지 **의식적으로 선택** 가능
> - 의식화 안 되면: 패턴이 무의식적으로 반복 (반복강박)

## 42. 합충 우선순위 (탐합망충) 🟢확립

합과 충이 동시에 존재할 때, **어느 쪽이 우선하는가**를 판별.

함수: `resolveHapChungPriority(relations)`
- **탐합망충**: 합이 인접하고 충이 멀면 → 합이 충을 무력화
- **충파합**: 충이 강하면 → 합이 깨짐
- AI 프롬프트에 주의사항으로 전달: "합과 충이 동시에 존재할 때: 인접한 합이 충을 해소하는지, 충이 합을 깨뜨리는지 판단하여 유기적으로 해석할 것"

### 42-1. 합충 역학 — 에너지 경합 3변수 이론 🟡유력

> **변수 1: 거리 (Proximity)** — 인접한 관계가 먼 관계보다 우선 (게슈탈트 근접성 원리)
> **변수 2: 기세 (Momentum)** — 대운/세운의 외부 에너지가 원국 내부보다 강할 수 있음
> **변수 3: 순도 (Purity)** — 1:1은 순수하고 강함. 투합(2:1)은 경쟁으로 분산

### 42-2. 합충의 심리적 귀결 🟡유력

> **합 우세 → 표면적 안정, 잠재적 긴장.** 눌린 충은 사라지지 않고 잠복.
> → 합이 약해지면(대운에서 합을 깨는 에너지가 오면) 눌려있던 충이 한꺼번에 터짐 = **지연된 폭발**
>
> **충 우세 → 표면적 갈등, 잠재적 변화.** 깨진 합은 새로운 형태로 재결합 가능.
> → 대운이 합을 지원하면 **재통합**

## 46. 십신 10개 정밀 매핑 🟡유력

| 십신 | 심리학 대응 | MBTS 해석 |
|------|------------|-----------|
| **비견** | 건강한 자아 동일시 | 동지. 주기능 확인 |

| **편재** | 도전적 현실 적응 | 공격적 현실 정복 경향 |
| **정재** | 안정적 현실 적응 | 꾸준한 축적 경향 |
| **편관** | 외부 위협 | 예측 불가 압박 경향 |
| **정관** | 구조적 규율 | 예측 가능한 질서 |
| **편인** | 비정규 학습 | 비주류 지식/직관적 깨달음 |
| **정인** | 정규 학습 | 체계적 교육/안전한 성장 |

**핵심:** 편(偏)은 강렬/예측 불가. 정(正)은 안정/예측 가능. 둘 다 필요하며 비율이 중요.

# PART XIX. MBTS의 궁극적 목표

## 58. 상관의 창의성 심리학 🟡유력

| 구조 | 창의성 유형 경향 |
|------|----------------|
| 상관 + 인성 | 파괴적 창조. 혁신가 |
| 식신 + 재성 | 실용적 창조. 크리에이터 |
| 화개살 + 식상 | 예술적 창조 |
| 식상 부재 | 잠재. 대운에서 폭발 가능 |

## 59. 재물운의 심리학 🟡유력

| 재성 구조 | 돈에 대한 경향 |
|-----------|--------------|
| 정재만 | 안전 추구. 저축형 |
| 편재만 | 게임 감각. 투자형 |
| 재성 없음 | 돈보다 의미. 대운에서 골든타임 |
| 재성 과다 | 물질 집착 경향 |
| 비겁탈재 | 유출 경향. 동업/보증 주의 |
| 식상생재 | 재능의 열매. 창작형 |

## 60. 대운 전환 시 심리 현상 체크리스트 🟡유력

### 용신 대운 진입 경향
- □ 갑자기 의욕이 샘솟음
- □ "뭔가 될 것 같다"는 확신
- □ 오래된 습관을 자연스럽게 버림
- □ 기회가 "우연히" 찾아옴

### 기신 대운 진입 경향
- □ 이유 없는 피곤과 의욕 상실
- □ 예전에 잘 하던 것이 안 됨
- □ "내 인생 이게 맞나?" 의문
- □ 건강 이슈 발생

### 교운기 (대운 전환 직전 ±2년) 경향
- □ 모든 게 정체된 느낌
- □ "뭔가 바뀌려는데 뭔지 모르겠다"
- □ 과거의 사람/사건이 떠오름

## 61. 자기 이해를 넘어 자기 초월로 🟢확립

모든 MBTS 풀이는 3단계로 끝나야 한다:

**1. 인식 (Recognition)** — "당신의 구조는 이런 경향을 보여줍니다"
**2. 수용 (Acceptance)** — "이건 좋거나 나쁜 게 아닙니다"
**3. 초월 (Transcendence)** — "이 구조를 알기에 넘어설 수 있습니다" + 구체적 행동

# 부록 B: MBTS 풀이 4단계 구조 🟢확립

### Stage 1: 원국 해부 (WHO)
4주 인지기능 매핑, 오행/음양/조후, 합충형해, 신살/공망, 격국/종격

### Stage 2: 역학 해석 (HOW)
십신 10개 역학, 5신 처방, 통변 워크플로우, 투출, 상생상극

### Stage 3: 시간축 분석 (WHEN)
대운(간지 분리/순역/간합 변환), 세운, 월운, 교운기, 합 트리거, 공망 해소

### Stage 4: 실전 통합 (WHAT) — 진단 50% + 성장 처방 50%
불일치 분석, 직업/연애/재물 타이밍, 건강, 궁합, 킬링 포인트
**처방은 반드시 구체적 행동을 포함**
